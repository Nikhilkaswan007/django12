# Paradise Nursery - Shopping Application

A beautiful React-based shopping application for browsing and purchasing houseplants.

## Features
- **Landing Page**: Stunning entry point with company information.
- **Product Listing**: Categorized plants with "Add to Cart" functionality.
- **Shopping Cart**: Real-time updates of quantities and total costs.
- **Redux State Management**: Seamless synchronization across the application.
- **Responsive Design**: Works on all screen sizes.

## Technologies Used
- React
- Redux Toolkit
- Lucide React (Icons)
- CSS3 with Glassmorphism
- Google Fonts (Outfit)

## How to Run
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open your browser and navigate to the provided local address.
